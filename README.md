Publicação dos Certificados de Bootcamp Developer Python e complementares na Parceria DIO Suzano 

## Formação em Andamento e Desenvolvimento de Competências: 

O acadêmico de Engenharia de Computação do IFMS – Campus Três Lagoas, João Orlando Medeiros Silvestre, segue avançando em sua jornada tecnológica no Bootcamp Python Developer Suzano, promovido pela DIO.

O programa reúne 46 atividades entre cursos, mentorias e desafios práticos, focados no desenvolvimento de competências em programação e tecnologias modernas. Até o momento, João já concluiu 4 módulos (9% da trilha), incluindo:

### ✔️ Ambiente de Desenvolvimento e Primeiros Passos com Python

Além da evolução técnica (Hard Skills), o bootcamp também contribui para o fortalecimento de Soft Skills, como pensamento lógico, resolução de problemas, organização, colaboração em projetos e aprendizagem contínua — competências essenciais para o mercado de tecnologia.

Com previsão de conclusão em 22 de junho de 2026, a formação inclui projetos práticos e mentorias que irão consolidar conhecimentos em Python, GitHub e desenvolvimento de software, preparando o estudante para os desafios da transformação digital.
